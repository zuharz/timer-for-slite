How to Embed the Countdown Timer in Slite

To embed this countdown timer in your Slite document, you can use the following code:

<script>(function(d, s, id) { 
  var js, pjs = d.getElementsByTagName(s)[0]; 
  if (d.getElementById(id)) return; 
  js = d.createElement(s); js.id = id; 
  js.src = "https://zuharz.github.io/timer-for-slite/loader.js"; 
  pjs.parentNode.insertBefore(js, pjs); 
}(document, "script", "slite-timer-sdk"));</script>
<a data-type="countdown" class="slite-timer" data-duration="300" style="display:block; width:100%; height:0; position:relative; padding-bottom:25%; margin:0 auto;" title="Countdown Timer" href="https://zuharz.github.io/timer-for-slite/">Countdown Timer</a>
Copied!

Customization Options

You can customize the timer by changing these attributes:

  • data-duration: Set the duration in seconds (default: 300 = 5 minutes)
  • title: Change the title displayed above the timer

To use this timer in Slite:
