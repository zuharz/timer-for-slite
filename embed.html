How to Embed the Countdown Timer in Slite

Based on Slite's embedding capabilities, here are different ways to embed this timer:

Option 1: ID-Based URL (Recommended)

Simply paste this URL in your Slite document for best compatibility:

https://zuharz.github.io/timer-for-slite/widget/countdown/1234
Copied!

Note: This format matches exactly how TickCounter embeds timers in Slite.

Option 2: Widget URL (Parameter-based)

Use this URL to customize the timer directly:

https://zuharz.github.io/timer-for-slite/widget/countdown/?duration=300&title=My%20countdown
Copied!

Customize: Change the number after "duration=" (in seconds) and add your own title after "title=".

Option 3: Slite-Compatible Embed Code

If direct links don't work, paste this HTML code:

<div class="position-relative radius-large shadow-inset-primary-on-selected border-small border-shade-45">
  <div class="radius-large overflow-hidden">
    <div style="left: 0; width: 100%; height: 0; position: relative; padding-bottom: 25%;">
      <iframe src="https://zuharz.github.io/timer-for-slite/widget/countdown/1234" 
        style="top: 0px; left: 0px; width: 100%; height: 100%; position: absolute; border: 0px; background-color: var(--shade-50);" 
        allowfullscreen=""></iframe>
    </div>
  </div>
</div>
Copied!

Customization Options

You can customize the timer by changing these parameters:

  • duration: Set the duration in seconds (default: 300 = 5 minutes)
  • title: Change the title displayed above the timer (URL encoded)

Using in Slite

Follow these steps for embedding in Slite:

  1. In your Slite document, simply paste the ID-Based URL (Option 1)
  2. Slite should automatically recognize and embed the timer as it does with TickCounter
  3. If needed, you can create multiple timers by using different IDs

If you encounter any issues, please open an issue on the GitHub repository.
